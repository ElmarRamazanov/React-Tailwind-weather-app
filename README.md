# 🌦️ Weather App (React + Tailwind)

Modern ve responsive bir web hava durumu uygulaması.
Kullanıcıların şehir arayarak anlık hava durumunu görüntülemesini sağlar.

Uygulama, gerçek zamanlı hava verilerini **OpenWeather API** üzerinden çekmektedir.

## ✨ Features

* 🌍 Şehir arama ile hava durumu sorgulama
* 🌡️ Anlık sıcaklık bilgisi
* 💨 Rüzgar hızı ve hissedilen sıcaklık
* ☁️ Hava durumu ikonları
* 📱 Mobil uyumlu responsive tasarım
* ⚡ Hızlı ve modern UI

---

## 🛠️ Tech Stack

* **React** – UI geliştirme
* **Vite** – hızlı geliştirme ortamı
* **Tailwind CSS** – stil ve responsive tasarım
* **Weather API** – hava verileri

---

## ⚙️ Installation

Projeyi yerelde çalıştırmak için:

```bash
# repoyu klonla
git clone https://github.com/KULLANICI_ADI/REPO_ADI.git

# klasöre gir
cd REPO_ADI

# bağımlılıkları yükle
npm install

# uygulamayı başlat
npm run dev
```

---

## 🔑 Environment Variables

API key kullanıyorsan `.env` dosyası oluştur:

```env
VITE_WEATHER_API_KEY=your_api_key_here
```

---

## 🧪 Build

Production build almak için:

```bash
npm run build
```
